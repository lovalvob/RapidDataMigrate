Welcome to Rapid Data Migrate.  A library and toolset for the rapid migration of data from traditional relational platforms to cloud plaforms.

The product is composed of multiple projects under a single solution in Visual Studio.  The core funcitonality is in the library (C# .net 9 as of this writing) which can be used from:
 * The command line (existing client)
 * A GUI written in Windows Forms (existing)
 * Powershell
 * Anything that can connect to a .net library

Note: Since this is an existing product that it being open-sourced under a new name, we're we're starting version 4.01
